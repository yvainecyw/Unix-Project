gcd.c

1. 操作模式：

在指令列輸入 vi gcd.c：利用 vi 指令編輯 gcd.c 檔
用 vi 編輯（i、x）程式碼，存檔之後離開（:wq!）
在指令列輸入 gcc -c gcd.c
在指令列輸入 gcc -o gcd.exe gcd.c
在指令列輸入 ./gcd.exe
輸入兩個數字求得答案

2. 執行結果：

輸入兩數:4 12
GCD:4


#include <stdio.h>

int gcd(int m,int n){
	while(n!=0){
		int r=m%n;
		m=n;
		n=r;
	}
	return m;
}

int main(int argc, char *argv[]) {
	int m,n;
	printf("輸入兩數:");
	scanf("%d %d",&m,&n);
	printf("GCD:%d\n",gcd(m,n));
	return 0;
}
